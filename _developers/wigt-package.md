# Wigt Package Structure Specfics

Wigt packages rely heavily on files being in an standardized directory structure. Here it is:

```shell
├── /_score-modules
│   └── /score_module.php
├── /assets
├── /img
│   ├── /screen-shots
│   │   ├── 1-thumb.png
│   │   ├── 1.png
│   │   ├── 2-thumb.png
│   │   ├── 2.png
│   │   ├── 3-thumb.png
│   │   └── 3.png
│   ├── icon-60.png
│   ├── icon-60@2x.png
│   ├── icon-92.png
│   ├── icon-92@2x.png
│   ├── icon-275.png
│   ├── icon-275@2x.png
│   ├── icon-394.png
│   └── icon-394@2x.png
├── demo.json
├── install.yaml
├── creator.html
├── scoreScreen.html
└── player.html
```
